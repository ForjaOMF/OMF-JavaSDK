GetSMSList

Retrieves a list of SMS messages via POP server

List GetSMSList(String Server, String User, String Pwd)

Parameters
Server
  String with POP server address

User
  String with user's login

Pwd
  String with user's password

Return
List of SMS messages in the form:

Each element in the list is a Message.

Below you can see a sample code to traverse the list: 
	Iterator mails = mailList.iterator();
	Message message =  null;

		message = (Message) mails.next();
